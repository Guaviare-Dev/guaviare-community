# Código de Conducta | GuaviareDev

Como miembros de **GuaviareDev**, nos comprometemos a crear un ambiente inclusivo, respetuoso y colaborativo para todos.

## 🤝 Principios

- Sé amable y respetuoso con los demás.
- Escucha activamente y valora las opiniones diversas.
- Evita lenguaje ofensivo, discriminatorio o agresivo.
- Aporta con humildad y disposición a aprender.

## 🚫 Comportamientos no aceptados

- Insultos, acoso o ataques personales.
- Discriminación por género, raza, orientación sexual, religión o nivel técnico.
- Spam o autopromoción excesiva.

## 📢 Reportes

Si presencias o experimentas una conducta inapropiada, puedes reportarla de forma confidencial a los administradores del repositorio o por correo (si se habilita).

## 🛠️ Aplicación

Los moderadores de GuaviareDev tienen la facultad de advertir, suspender o expulsar a quienes incumplan este código.

---
Este código se inspira en el [Contributor Covenant](https://www.contributor-covenant.org/es/version/2/0/code_of_conduct/).  
Juntos construimos una comunidad segura y enriquecedora.
